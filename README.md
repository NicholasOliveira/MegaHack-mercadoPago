## **Quick-start**

1. Instalar Yarn
   `npm install yarn -g`

2. Baixar dependencias
   `yarn`

3. Startar Servidor Dev
   `yarn start`

4. Rodar Build de Produção
   `yarn build`

5. Rodar Suite de Testes
   `yarn test`

6. Rodar Linter
   `yarn lint`

7. Rodar o Formatador de Código
   `yarn format`

8. Inicializar Ferramenta de Inspeção de Bundle
   `yarn bundle-profile:analyze`
